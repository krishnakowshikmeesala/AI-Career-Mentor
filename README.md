AI Career Mentor

AI Career Mentor is a web-based application developed using Django and Google's Gemini AI.
The platform helps students discover suitable career paths based on their interests, strengths,
learning preferences, career goals, and personal challenges.

Features
-> Student Career Assessment Form
-> AI-Powered Career Recommendations
-> Personalized Career Analysis Report
-> Career Match Score
-> Skill Recommendations
-> Six-Month Learning Roadmap
-> Free Learning Resource Suggestions
-> Modern and Responsive User Interface

Technologies Used
-> Python
-> Django
-> HTML5
-> CSS3
-> Bootstrap
-> SQLite
-> Google Gemini AI API


How It Works
Students fill out a career assessment form.
The system collects information such as:
-> Interests
-> Strengths
-> Career Goals
-> Learning Preferences
-> Challenges
-> Dream Career
Gemini AI analyzes the data.
A personalized career report is generated containing:
-> Recommended Career
-> Career Match Score
-> Career Explanation
-> Required Skills
-> Six-Month Roadmap
-> Learning Resources

Project Objective
The goal of this project is to provide students, especially those with limited career guidance,
with AI-powered career recommendations and a clear learning roadmap to help them achieve their professional goals.
